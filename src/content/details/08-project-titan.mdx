**Koncepcja**
Project TITAN to manifest moich umiejętności. Nie jest to szablonowe portfolio, lecz aplikacja webowa klasy "Elite". Design czerpie inspirację z interfejsów sci-fi i wojskowych systemów operacyjnych.

**Stos Technologiczny (Bleeding Edge)**
* **Meta-Framework:** Astro 5.0 (Content Layer API) do zarządzania danymi.
* **UI Logic:** Svelte 5 (Runes) – nowy, reaktywny model stanu, który wyprzedza Reacta pod względem wydajności.
* **Styling:** Tailwind CSS v4 (Alpha) – wykorzystanie natywnego silnika CSS dla maksymalnej szybkości budowania.
* **3D & FX:** Customowe shadery i transformacje CSS 3D (Exploded View) dla kart projektów.

**Efekt Końcowy**
One-Pager, który waży mniej niż przeciętne zdjęcie na Instagramie, a oferuje doświadczenie aplikacji natywnej. Pełna dostępność (A11y), obsługa klawiatury i wynik 100/100 w testach wydajnościowych. To dowód, że "piękne" nie musi znaczyć "wolne".
